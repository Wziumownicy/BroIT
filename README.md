# BroIT
Symulator sklepu internetowego

Co to BroIT?

BroIT to sklep internetowy na którym kupisz wziuma! 

# Instalacja

Arch Linux

```
git clone https://github.com/Wziumownicy/BroIT.git
cd BroIT && makepkg -si
```
Pozostałe dystrybucje

```
git clone https://github.com/Wziumownicy/BroIT.git
cd BroIT && chmod +x BroIT_installer.sh
./BroIT_installer.sh
```
